
앞에서는 기존에 있는 웹사이트에 React를 적용하는 방법에 대해서 배워봤다
하지만 새로운 웹사이트를 만들 때는 굳이 그렇게 할 필요가 없다  -->  처음부터 React가 적용되어있는 상태로 개발을 진행하면 된다

React로 웹사이트를 개발할 때에는 주로 create-react-app 이란 것을 사용한다

create-react-app (CRA) : 리액트 웹 에플리케이션을 개발하는데 필요한 모든 설정이 되어있는 상태로 프로젝트를 생성해주는 도구

CRA 은 npx 명령어를 통해서 실행할 수 있다
사용법 : $ npx create-react-app <your-project-name>
실제 사용 예제 : $ npx create-react-app my-app

실행한 후에 터미널 마지막 부분을 보면 에플리케이션을 실행할 수 있는 명령어를 친절하게 알려준다
cd my-app (경로 변경)
npm start (애플리케이션 실행)

--> 명령어를 실행하고 기다리면 자동으로 브라우저가 열리면서 localhost:3000 으로 접속하게 된다
--> 여기서 localhost 라는 것은 현재 내가 사용하고 있는 컴퓨터를 의미한다 (React 애플리케이션이 로컬 개발환경에서 실행된 것을 의미한다)
