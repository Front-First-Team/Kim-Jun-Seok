
HTML : Hyper Text Markup Language
--> 마크업 언어의 한 종류이다

Markup 이란 문서나 데이터를 처리하기 위해 "문서에 추가되는 정보" 를 의미한다

HTML : 웹사이트의 뼈대를 구성하기 위해 사용하는 마크업 언어

수많은 페이지가 존재하는 복잡한 웹사이트인 경우 HTML 파일이 수십에서 수백개가 될텐데, 이 많은 파일을 어떻게 다 관리할까?
--> 이 문제를 해결하기 위해 나온 것이 바로 SPA이다

SPA : Single Page Application  (하나의 페이지만 존재하는 웹 사이트 or 웹 어플리케이션)
--> 페이지가 하나라는 것은, HTML 파일이 하나라는 것이다
(각각 다른 컨텐츠, 컴포넌트들이 페이지마다 동적으로 body 태그 안에 내용물을 채워넣게 된다  -->  React가 내용물을 채워넣는다고 보면 된다)

CSS : Cascading Style Sheets
--> 웹사이트의 레이아웃, 글꼴, 색상 등의 디자인을 입히는 역할을 한다
--> HTML로 웹사이트의 구조를 만들었다면, 그 위에 CSS로 디자인을 입혀야 아름다운 웹사이트가 완성이 된다
