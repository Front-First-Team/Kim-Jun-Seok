
React : 사용자 인터페이스를 만들기 위핸 자바스크립트 라이브러리 (자바스크립트 기능 모음집)

라이브러리(도서관) : 자주 사용되는 기능들을 정리해 모아 놓은 것

사용자 인터페이스 : User Interface (UI)
--> 사용자와 컴퓨터 프로그래밍이 서로 상호작용을 하기 위해 중간에서 서로간의 입력, 출력을 제어해주는 것이 UI 이다
--> ex) 버튼, 텍스트 입력창 등

자바스크립트 UI 라이브러리 : 화면을 만들기 위한 기능들을 모아 놓은 것
--> 라이브러리 ex) React
--> 프레임워크 ex) NGULAR JS, Vue.js

라이브러리는 제어 권한이 개발자에게 있고, 프레임워크는 제어 권한이 프레임워크 자기 자신에게 있다

웹 개발의 트렌드 : 한때는 앤귤러 JS 가 웹 개발의 정답인 것처럼 여겨지는 시기가 있었다 -> 한가지 기술이 무조건 평생 대세가 될 것이라 생각하면 안된다
                             기술의 큰 흐름을 읽는 것이 중요하다
                             웹사이트의 작동 원리와 흐름을 함께 이해하는 것이 중요하다
